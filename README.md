# Installing

Development:

```
$ git clone https://github.com/duosecurity/duo_api_ruby.git
$ cd duo_api_ruby
```

System:

```
$ gem install duo_api
```

Or add the following to your project:

```
gem 'duo_api', '~> 1.0'
```
